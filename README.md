# SENTRI – Chatbot AI Edukasi Keuangan

**SENTRI** (Sistem Edukasi dan Notifikasi Terintegrasi) adalah web app chatbot AI yang dirancang untuk:

- Memberikan edukasi seputar keuangan digital
- Mendeteksi konten penipuan & judi online
- Membantu masyarakat mengenali risiko investasi palsu dan platform ilegal

---

## ✨ Fitur Utama

- ✅ Chatbot AI berbasis JavaScript
- ✅ Deteksi kata kunci penipuan dan perjudian
- ✅ Edukasi interaktif langsung dari browser
- ✅ Desain ringan dan mudah diakses

---

## 🧠 Teknologi Digunakan

- HTML, CSS, JavaScript
- GitHub Pages (untuk hosting)
- Model logika dasar berbasis keyword + AI assistant (pengembangan lanjutan)
